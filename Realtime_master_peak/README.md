# 概要
* 音の出力デバイスを選択すると、そのデバイスで流れている音のリアルタイムな音量が表示される。

* PCのボリュームが「0」でも何かが再生されていれば、その音量が表示される。
* 複数のタブで再生されいても、上書きされるだけでエラーにはならない。
